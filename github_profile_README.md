# Hi, I'm Raziye Aghapour 👋

PhD candidate in Industrial Engineering at UT Arlington (expected December 2026), 
researching large-scale power systems planning and optimization. My work sits at 
the intersection of mathematical programming, computational modeling, and data science 
— building Python workflows to automate complex simulations and applying machine 
learning to extract insights from high-dimensional energy system outputs.

---

## What I Work On

- **Energy system modeling** — building and running large-scale scenarios in NREL ReEDS 
  and GCAM-USA to study renewable integration, transmission expansion, and cost trade-offs 
  in the ERCOT grid
- **Optimization** — formulating and solving LP, MIP, and stochastic optimization problems 
  using Gurobi and GAMS for infrastructure planning under uncertainty
- **Python data pipelines** — automating input preprocessing, batch execution, and 
  post-processing across hundreds of simulation cases
- **Applied ML** — using scikit-learn, XGBoost, and SHAP to build predictive models 
  on simulation output data and identify key planning drivers
- **OR methods in practice** — applying operations research techniques to real-world 
  problems in energy, buildings, and transportation systems
